# Dise-o_Normalizaci-n_e_Implementaci-n_de_Bases_de_Datos_Relacionales-
Demostrar la capacidad de transformar datos no estructurados (JSON/CSV) en un modelo  relacional óptimo mediante el proceso de normalización (1NF a 4NF/BCNF) y su  implementación funcional en un entorno de desarrollo profesional (kaggle + SQLite).
